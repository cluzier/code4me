
# Code4Me
CEN4350C Final Project

## Installation
Install dependencies

```bash
  cd code4me
  npm install
```
    
## Deployment
To deploy this project run

```bash
  npm run start
```

  
## Tech Stack
**Client:** React, TailwindCSS

**Server:** Node, Express

**API:** Stripe, Commerce

**[Dependencies](https://github.com/cluzier/code4me/network/dependencies)**
